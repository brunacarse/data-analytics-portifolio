# 📊 Financial Planning & Analysis Dashboard

## 👤 Author

Bruna Carse

---

## 🧠 Business Context

This project simulates a Financial Planning & Analysis (FP&A) scenario where business leaders need visibility into financial performance, budget execution and profitability.

The objective is to transform financial data into actionable insights to support strategic decision-making.

---

## 🎯 Objective

The analysis aims to answer:

* How is revenue performing against budget?
* Which areas are driving financial deviations?
* What are the main cost drivers?
* How is profitability evolving over time?

---

## 📦 Dataset

Synthetic financial dataset created for portfolio purposes.

Period:
2024–2025

Main metrics:

* Revenue
* Expenses
* Budget
* Profit
* Profit Margin

---

## 🧹 Data Preparation

Steps performed:

* Data cleaning

* Data transformation
* KPI calculation
